# TODO - Melhorias na Sidebar Zênite360

## Objetivo
Reorganizar a sidebar para melhor usabilidade, organização visual e navegação intuitiva.

## Problemas Identificados
- [ ] Itens de menu organizados de forma linear sem agrupamento lógico
- [ ] Falta de submenus para funcionalidades relacionadas
- [ ] Ausência de indicadores visuais (notificações, status)
- [ ] Navegação pode ser confusa com muitos itens

## Melhorias Propostas

### 1. Reorganização em Grupos Lógicos
- [ ] **Área Clínica**: Dashboard, Pacientes, Agendamentos, Triagem, Consultas, Internamento
- [ ] **Área Técnica**: Farmácia, Laboratório, Manutenção, Patrimônio
- [ ] **Área Administrativa**: RH, Financeiro, Secretaria, Serviços Gerais, Casa Mortuária
- [ ] **Sistema**: Configurações

### 2. Submenus Expansíveis
- [ ] RH: Funcionários, Departamentos, Escalas
- [ ] Financeiro: Faturas, Resumos
- [ ] Patrimônio: Ativos, Gases, Manutenção
- [ ] Serviços Gerais: Contratos, Resíduos, Cozinha, Funcionários

### 3. Indicadores Visuais
- [ ] Separadores visuais entre grupos
- [ ] Ícones mais descritivos e consistentes
- [ ] Cores diferenciadas por grupo (opcional)

### 4. Melhorias de UX
- [ ] Animações suaves para expansão de submenus
- [ ] Indicadores de estado (expandido/colapsado)
- [ ] Tooltips informativos

## Implementação

### Fase 1: Estrutura Básica

- [ ] Adicionar separadores visuais

### Fase 2: Funcionalidades
- [ ] Estado de expansão/colapso
- [ ] Navegação ativa correta
