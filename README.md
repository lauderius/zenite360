## Hospital360: ERP Hospitalar Integrado (Next.js + MySQL)

**Zênite360** é uma solução de gestão hospitalar de missão crítica, desenvolvida para unificar a logística, o atendimento clínico e a gestão de infraestrutura em uma única plataforma em tempo real.

O diferencial do sistema é o seu **Motor de Dependência de Recursos**, que impede conflitos de agenda ao cruzar a disponibilidade de profissionais, equipamentos médicos e insumos farmacêuticos simultaneamente.
